*** Apenas esboço / exemplo

1) 	Criar a pasta HDFS /etc/resultado

2) 	Copiar o arquivo do libble "mskgrafico.ods" para este local 

3)	Copiar os arquivos "retorno_gr1.csv" e "retorno_gr2.csv" neste local 

4) 	Colocar no arquivo "analisar-sentimento.py" mais 
	duas saidas para "retorno_gr1.csv" e "retorno_gr2.csv", seguindo os formatos abaixo


* Observações: 

- A saída será feita pelo arquivo em python
- Colocar vírgula em vez de ponto
- É apenas um exemplo/esboço
- Os arquivos retornos são para servir de exemplo
- Ordem de Geração:
	A)Executar o arquivo python
	B)Abrir o arquivo "mskgrafico.ods"
- Atenção: O Gráfico fica na segunda planilha dentro do arquivo principal "mskgrafico.ods"
